# Skill Card Format Standard (v0.1)

Skills in Zhiné（知你）are not hardcoded string prompt templates. They represent modular, evaluatable, and composable capability nodes[cite: 1].

### Core Requirements:
1. **Scope Matrix**: Dictates the exact boundaries and execution criteria under which the skill activates[cite: 1].
2. **Success Metrics & Failure Modes**: Concrete evaluation keys used by the Experience Engine to attribute outcomes and update the asset state[cite: 1].
